
# Creating User Accounts and Managing Tasks in the Application

### Objective

This SOP outlines the steps to create user accounts, manage tasks, and navigate the application effectively.

### Key Steps

**1. Create User Accounts** [0:19](https://loom.com/share/4f33567a52ef415ab62bb35ec481d4f7?t=19)

![generated-image-at-00:00:19](https://loom.com/i/ca54b9993c7143f1818c4baa415674c7?workflows_screenshot=true)

- Access the application and navigate to the 'Create Account' page.
- Fill in the required fields: Name, Email, and Password.
- Submit the form to create a new user account.

**2. Sign In to the Application** [0:28](https://loom.com/share/4f33567a52ef415ab62bb35ec481d4f7?t=28)

![generated-image-at-00:00:28](https://loom.com/i/2a79a39cfab94e879f0e632106fd618a?workflows_screenshot=true)

- Go to the 'Sign In' page.
- Enter the registered Email and Password.
- Click 'Login' to authenticate the user.

**3. Navigate the Home Page** [1:03](https://loom.com/share/4f33567a52ef415ab62bb35ec481d4f7?t=63)

![generated-image-at-00:01:03](https://loom.com/i/18caaebd788047a6b80bd937bdff4659?workflows_screenshot=true)

- Upon successful login, you will be redirected to the Home Page.
- Familiarize yourself with the sections: Tasks, Products, Team, and Profile.

**4. View Team Overview** [1:10](https://loom.com/share/4f33567a52ef415ab62bb35ec481d4f7?t=70)

![generated-image-at-00:01:10](https://loom.com/i/adb68b6048594658949aec5788f9e921?workflows_screenshot=true)

- Check the total number of team members and their active status.
- Review the completion rate and pending tasks.

**5. Manage Pending Tasks** [1:29](https://loom.com/share/4f33567a52ef415ab62bb35ec481d4f7?t=89)

![generated-image-at-00:01:29](https://loom.com/i/eeba95e2f1f9444cb9013e280b2dacf3?workflows_screenshot=true)

- Identify the number of pending tasks assigned to team members.
- Click on the 'Team' section to view details of pending tasks.

**6. Create a New Task** [2:48](https://loom.com/share/4f33567a52ef415ab62bb35ec481d4f7?t=168)

![generated-image-at-00:02:48](https://loom.com/i/3da367893a8044d58a7e47f608f2964f?workflows_screenshot=true)

- Navigate to the 'New Task' section.
- Enter the Task Title and Description.
- Assign the task to a specific user.

**7. Send Notifications to Users** [2:58](https://loom.com/share/4f33567a52ef415ab62bb35ec481d4f7?t=178)

![generated-image-at-00:02:58](https://loom.com/i/6ef3e8385d104db2ba7e3ecf7be4dbfe?workflows_screenshot=true)

- Use the messaging feature to send notifications to users.
- Include a URL for users to join the application.

**8. Review Application Codebase** [3:29](https://loom.com/share/4f33567a52ef415ab62bb35ec481d4f7?t=209)

![generated-image-at-00:03:29](https://loom.com/i/f9d86014966a493ea025a203f201c3f8?workflows_screenshot=true)

- Access the codebase, which consists of two main folders: Dinescocra Backend and Dinescocra Mobile.
- Review the backend logic in the 'Dinescocra Backend' folder.

**9. Understand Database Configuration** [3:45](https://loom.com/share/4f33567a52ef415ab62bb35ec481d4f7?t=225)

![generated-image-at-00:03:45](https://loom.com/i/45fc405c302d44ba98383c44d67c35fa?workflows_screenshot=true)

- Locate the 'database.js' file in the config folder for database connection details.

**10. Explore Controllers** [4:03](https://loom.com/share/4f33567a52ef415ab62bb35ec481d4f7?t=243)

![generated-image-at-00:04:03](https://loom.com/i/ae51674b2c5e4f23bcea15043c4a061d?workflows_screenshot=true)

- Review the various controllers, including Authentication and Annotation controllers, for understanding application logic.

### Cautionary Notes

- Ensure that all user data is handled securely and in compliance with data protection regulations.
- Double-check email addresses for accuracy before sending notifications.

### Tips for Efficiency

- Use templates for task descriptions to save time when creating new tasks.
- Regularly update the team on pending tasks to maintain accountability.
